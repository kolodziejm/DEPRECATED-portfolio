O mnie

Jestem uczniem trzeciej klasy technikum informatycznego. Swoją przygodę z front-endem zacząłem w lutym tego roku i od tego czasu codziennie pogłębiam swoją wiedzę.

Jestem samoukiem, całą swoją wiedzę uzyskałem dzięki kursom, czytaniu dokumentacji oraz wykonywaniu projektów. Tworzenie sprawia mi przyjemność i daje satysfakcję, nie mam problemu z poświęcaniem dużej ilości czasu na rozwój. Obecnie rozwijam się głównie z zakresu JavaScripta.

Poza kodowaniem moim zainteresowaniem są gry strategiczne, w szczególności Starcraft oraz poker. Jestem również miłośnikiem sim-racingu oraz fanem motoryzacji.

Poszukuję stażu jako front-end developer na terenie Śląska.

Technologie które wykorzystuję

  • HTML5

    Buduję szkielet strony w oparciu o semantykę HTML5

  • CSS3

    Projektuję responsywny layout z wykorzystaniem flexboxa, CSS grid oraz podejścia mobile first.

  • JavaScript (ES6+)

    Manipulacja DOMem, wykorzystywanie zewnętrznych API, asynchroniczny JS

  • Sass

    Wykorzystuję preprocessor Sass dla wydajniejszej pracy z CSS

  • Bootstrap

    Bootstrap pomaga mi w budowie szybkich prototypów aplikacji

  • React

    Obecnie znam absolutne podstawy, lecz po dogłębnym poznaniu JSa nauka Reacta to mój priorytet

  • Git

    Znam podstawy Gita, potrafię wykonać podstawowe operacje
